### ✨ **Today Advice – 조언 카드 앱**
- 카드 3장 중 프리미엄 카드 1장은 보상형 광고로 열람 가능
- 모든 조언은 API 서버에서 동적 제공 → 콘텐츠 유연성 확보
- Google Play 비공개 테스트 2회, 피드백 기반 기능 개선 완료
- 사용자 참여 기반 반복 테스트 → 현재 프로덕션 승인 대기 중
